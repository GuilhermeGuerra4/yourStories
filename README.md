##  functionalities

- Write stories
- Read stories
- Like and comment stories
- See how many likes and comments my story have
- receive notifications when someone likes or comments on my story
- Login with google signin button

## stack used
- Flask
- Flask Socket-io
- React-native
- Sqlite3
- OneSignal api
- Google signin api
